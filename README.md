# Memor-oji
A web-based memory game with emojis.

##Usage
Run game.html in a web browser. Click cards and try to find matches!

Works best on modern desktop browsers such as Chrome, Firefox, or Safari.

##Main files

* /game.html
* /css/style.css
* /js/app.js

##Resources
